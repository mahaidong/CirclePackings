\frametitle{1. ~ Planar graphs}


A \emph{\bfseries graph} is a data structure consisting of:
\begin{itemize}[$\bullet$]
 \item A set of \emph{vertices}
 \item A set of \emph{edges} = relation between vertices
\end{itemize}
